# NQL Systems

NQL Systems is the public home for my work on human-governed AI systems, artifact-backed workflows, and bounded builder experiments.

The main system is **Addison**.

Addison is a local, daemon-backed state system built around persistent artifacts, deterministic context construction, and stateless model inference. It is designed so memory, continuity, and control belong to the system rather than to the model.

The main workflow is **WBTSR**  
(**Work, Build, Truth, Session, Rehydrate**).

WBTSR is the human-governed method I use to build with AI through bounded work, separate builder and auditor roles, explicit truth surfaces, and restart-safe rehydration.

## Public posture

This organization is intentionally partial.

Public repos here are curated surfaces:
- public-safe documentation
- bounded project mirrors
- experimental work, I am comfortable surfacing

They are not the full private development authority.

## Current focus
